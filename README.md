# Personal Website of Osteopath  

This repository contains the source files for a personal website of an osteopath based in Tel Aviv. 
The website provides information about available treatments, services, and contact methods.

## Features

- Information about osteopathic treatments
- Services offered
- Contact details and appointment booking
- Simple, responsive design

## Usage

- The website is hosted on GitHub Pages: `https://vlady64-afk.github.io/`
- You can edit HTML, CSS, and JavaScript files to customize the content and design
